# 🌤️ Weather Data Pipeline + AI Dashboard

## 📌 Description
This project is a full data pipeline that:
- collects real-time weather data from an API
- stores it in a SQLite database
- visualizes it with Streamlit
- applies machine learning to predict temperature

---

## 🚀 Features
- 🌍 Multi-city weather (Bruxelles, Paris, Berlin, Liège, Huy)
- 🗄️ SQLite database storage
- 📊 Interactive Streamlit dashboard
- 🧠 Machine Learning prediction (Linear Regression)
- 📈 Data visualization (matplotlib)

---
